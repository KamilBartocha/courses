# Prework - przygotowanie środowiska przed zajęciami

Wykonaj poniższe kroki **przed pierwszymi zajęciami**. Zajmie to ok. 15–20 minut.

---

## 1. Instalacja Pythona

### Windows

- [ ] Otwórz przeglądarkę i wejdź na **python.org/downloads**
- [ ] Kliknij **Download Python 3.x.x** (najnowsza wersja)
- [ ] Uruchom pobrany plik `.exe`
- [ ] ⚠️ Na pierwszym ekranie zaznacz **"Add Python to PATH"** (domyślnie odznaczone!)
- [ ] Kliknij **Install Now**
- [ ] Po instalacji otwórz **wiersz poleceń** (`Win + R` → wpisz `cmd` → Enter)
- [ ] Wpisz: `python --version` i naciśnij Enter
- [ ] ✅ Powinnaś/powinieneś zobaczyć np. `Python 3.12.4`

### macOS

- [ ] Otwórz przeglądarkę i wejdź na **python.org/downloads**
- [ ] Kliknij **Download Python 3.x.x** (najnowsza wersja)
- [ ] Uruchom pobrany plik `.pkg` i przejdź przez instalator (Next → Agree → Install)
- [ ] Otwórz **Terminal** (`Cmd + Spacja` → wpisz `Terminal` → Enter)
- [ ] Wpisz: `python3 --version` i naciśnij Enter
- [ ] ✅ Powinnaś/powinieneś zobaczyć np. `Python 3.12.4`

---

## 2. Instalacja Visual Studio Code

### Windows i macOS

- [ ] Wejdź na **code.visualstudio.com**
- [ ] Kliknij **Download** — strona automatycznie wykryje Twój system
- [ ] **Windows:** uruchom `.exe` i przejdź przez instalator (Next → Next → Install)
- [ ] **macOS:** otwórz pobrany `.zip`, przeciągnij `Visual Studio Code` do folderu **Aplikacje**
- [ ] Uruchom VS Code
- [ ] ✅ Powinnaś/powinieneś zobaczyć okno edytora z ekranem powitalnym

---

## 3. Instalacja rozszerzeń w VS Code

Rozszerzenia instalujesz przez panel boczny VS Code(ikona klocków).

**Jak otworzyć panel rozszerzeń:**
- Kliknij ikonę klocków po lewej stronie `⊞` lub naciśnij `Ctrl+Shift+X` (Windows) / `Cmd+Shift+X` (macOS)

### Rozszerzenia do zainstalowania

W polu wyszukiwania wpisz nazwę, kliknij wynik od właściwego autora i kliknij **Install**.

**Python (Microsoft)**
- [ ] Wyszukaj: `Python`
- [ ] Autor: **Microsoft**
- [ ] Kliknij Install

**Pylance (Microsoft)**
- [ ] Wyszukaj: `Pylance`
- [ ] Autor: **Microsoft**
- [ ] Kliknij Install

**Jupyter (Microsoft)**
- [ ] Wyszukaj: `Jupyter`
- [ ] Autor: **Microsoft**
- [ ] Kliknij Install

### Weryfikacja

- [ ] Utwórz nowy plik: `Ctrl+N` (Windows) / `Cmd+N` (macOS)
- [ ] Zapisz jako `test.py`: `Ctrl+S` / `Cmd+S`
- [ ] Wpisz w pliku: `print("Działa!")`
- [ ] Uruchom: kliknij przycisk ▶ (prawy górny róg) lub naciśnij `Ctrl+F5`
- [ ] ✅ W dolnym panelu powinnaś/powinieneś zobaczyć: `Działa!`
